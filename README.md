# rcinpy 

**rcinpy** is a lightweight Python bridge that allows you to execute C code directly from your Python scripts without manual compilation.

## Features
- Run raw C strings or `.c` files.
- Automatic compilation using `gcc`.
- Handles argument passing (auto-converts types to strings).
- Cleans up binaries automatically after execution.
